A Sonic Map, What's that?!


A sonic map is an interactive map that uses sound as a way to convey qualitative information on a specific location.

No location sounds exactly the same as the next one.

This website serves as a sort of "geo-sonic database" that user can add on to by uploading files (you should try it!).

Each soundscape is unique and deserves to be documented.

While you're here, come listen to the world!
